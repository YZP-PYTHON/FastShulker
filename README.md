# FastShulker

FastShulker 是一个基于 **Paper/leaves 1.21.3** 的轻量级服务器插件，允许玩家 **手持潜影盒对空气右键即可直接打开潜影盒 GUI**，无需将其放置在地面上。



---

## ✨ 功能介绍

- 🖱 **右键空气直接打开潜影盒**
    - 玩家手持任意颜色的潜影盒，对空气右键即可打开其内容
    - 不影响对方块右键放置潜影盒的原版行为


- 🧩 **兼容 CoreProtect**
    - 不使用异常的虚拟方块放置
    - 不会触发 CoreProtect 报错或异常日志
    - 潜影盒内容修改行为符合正常物品数据变更逻辑

---

## 🖥 运行环境

- **Minecraft 服务端**
    - Paper/leaves **1.21.3**
    - 不保证兼容 Spigot / Bukkit（推荐使用 Paper）

- **Java 版本**
    - Java **17** 或更高（与 Paper 1.21.x 要求一致）

- **依赖插件**
    - 无强制依赖
    - 可选兼容：CoreProtect
